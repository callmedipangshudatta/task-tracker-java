# Task Tracker CLI

A simple command-line Task Tracker built using Java.

## Features

- Add Task
- View Tasks
- Mark Task Completed
- Delete Task
- File Storage using tasks.txt

## Technologies

- Java
- OOP
- ArrayList
- File Handling

## Project Structure

Main.java
Task.java
TaskManager.java
tasks.txt

## Future Improvements

- JSON Storage
- Search Task
- Priority System
- Sorting Tasks
